# Nikel - Sistema de Controle Financeiro

Projeto desenvolvido durante o **Codai 2.0**, programa de iniciação à programação da Growdev.

## Sobre o Projeto

O Nikel é uma aplicação web simples para controle financeiro pessoal, permitindo aos usuários organizarem suas finanças de forma fácil e prática.

## Funcionalidades

- **Login**: Sistema de autenticação de usuários
- **Home**: Painel principal com visão geral das finanças
- **Transações**: Registro e controle de receitas e despesas

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5.1.3
- JavaScript

## Objetivo do Projeto

Este é um projeto educacional voltado para iniciantes em programação, com foco em:
- Desenvolvimento de interfaces web
- Manipulação de formulários
- Estruturação de páginas HTML
- Estilização com CSS e Bootstrap
- Lógica de programação com JavaScript
- Manipulação do DOM (Document Object Model)
- Validação de dados
- Interatividade e dinamismo nas páginas
